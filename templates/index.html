My App

An end-to-end machine learning pipeline is developed to predict students' math performance using seven independent variables. Exploratory analysis revealed that students' performance correlated with race/ethnicity, lunch type, and parental education background. Additionally, it was noted that females outperformed males. While enrollment in a test preparation course showed limited correlation, completing the course proved beneficial for students.
